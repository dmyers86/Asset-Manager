Google Workspace Admin — Chromebook Sync

Import a ChromeOS device export directly from your Google Admin console. Device serial numbers, asset IDs, status, assigned user, and org unit are all mapped automatically.

How to export from Google Admin:
  1. Go to admin.google.com
  2. Navigate to Devices → Chrome → Devices
  3. Select all devices (or filter by org unit first)
  4. Click More → Download CSV
  5. Drop the downloaded CSV below
Columns mapped automatically: Serial Number → Serial  ·  Asset Id → Asset #  ·  Status → Status  ·  User → Assigned user  ·  Org Unit → Notes  ·  Last Sync → Activity
